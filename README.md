# Programming Paradigms — CS3022, NPS

Labs and lectures from CS3022 (Programming Paradigms) at NPS. Covers language design, parsing, type systems, and multiple programming paradigms.

## Labs

| Lab | Topic | Language |
|-----|-------|----------|
| Lab 0 | Hello World setup | Batch |
| Lab 1 | Lexical analysis — ANTLR grammar for email filtering, finite state machines | ANTLR/BNF |
| Lab 2 | Logic programming — obfuscated Prolog | Prolog |
| Lab 3 | Object-oriented design — animal guessing game with binary tree | C++ |
| Lab 4 | Systems programming — Crabby web server | Rust |
| Lab 6 | Functional programming | Python |

## Lectures

Weekly lecture slides covering: syntax, scanning/parsing, binding, object lifetime, type systems, pointers/GC, imperative programming with Rust, and more.

## Technologies

ANTLR · Prolog · C++ · Rust · Python
